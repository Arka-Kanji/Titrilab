**TitriLab** is a comprehensive, interactive, web-based chemistry titration simulator. It allows students, educators, and chemistry enthusiasts to simulate various quantitative analysis techniques, including acid-base, redox, and advanced instrumental titrations, complete with real-time dynamic graphing and accurate chemical calculations.

## ✨ Features

### 🧪 Multi-Mode Simulation
* **Acid-Base Titrations:** Choose from 8 acids (including strong, weak, and polyprotic like $H_3PO_4$ and $H_2SO_4$) and 6 bases. Includes compatibility warnings and 6 different pH indicators (Phenolphthalein, Methyl Orange, etc.).
* **Redox Titrations:** Permanganometry ($KMnO_4$), Dichrometry ($K_2Cr_2O_7$), and Cerimetry ($Ce^{4+}$).
* **Advanced & Instrumental:** Simulate Complexometry (EDTA for water hardness/metal estimation), Iodometry, Argentometry (Mohr, Volhard, Fajans), and Conductometry.

### 🔬 Interactive Laboratory View
* **Animated Apparatus:** Visual burette draining, dropping animations, and a magnetic stirrer.
* **Dynamic Color Changes:** Flask solution colors accurately reflect real-world indicator transitions and reaction endpoints.
* **Live Meters:** Real-time digital readouts for pH, Potential $E$ (mV), and Conductance.

### 📈 Live Data Graphing
* **Dynamic Curves:** Real-time plotting of the titration curve as titrant is added.
* **Interactive Controls:** Start, pause, reset, and adjust drop rates (up to 2.0 mL/s).
* **Export:** One-click download of the generated titration curve as a `.png` image for lab reports.

### 📚 Built-In Educational Resources
* **Theory Tab:** In-depth chemical principles, balanced equations, and Nernst/Henderson-Hasselbalch derivations for every titration type.
* **Procedure Tab:** Step-by-step laboratory protocols, apparatus lists, and safety warnings.
* **Calculations Tab:** Interactive calculators for Molarity, Equivalence Point, pH, Buffer Capacity, EDTA pMetal, Water Hardness, and more. *Bonus: Automatically imports equivalence point data from your recent simulation!*

### 🌗 UI & Accessibility
* **Theming:** Supports Light, Dark, and System Auto themes.
* **Responsive Design:** Fully responsive layout built with modern CSS.
* **Immersive Experience:** Includes toggleable background laboratory lo-fi music.

## 🚀 Getting Started

TitriLab is built using pure **Vanilla HTML, CSS, and JavaScript**. No build tools, bundlers, or package managers are required!
